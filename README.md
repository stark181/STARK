# PromptBase

AIを使いこなす実践知が集まるプラットフォーム。  
「誰かが実際に試してうまくいった知見」を核にした、ビジネスパーソン向けプロンプト共有サービス。

## ローカル起動手順

### 必要環境
- Node.js 18以上
- npm 9以上

### 起動

```bash
cd prompt-platform
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## ディレクトリ構成

```
prompt-platform/
├── app/
│   ├── layout.tsx           # 共通レイアウト・メタデータ
│   ├── page.tsx             # 一覧ページ（検索・フィルター・ソート）
│   ├── globals.css
│   └── prompts/
│       └── [id]/
│           └── page.tsx     # 詳細ページ（プロンプト本文・レビュー・Fork）
├── components/
│   ├── Header.tsx           # ナビ・検索バー
│   ├── FilterBar.tsx        # カテゴリ・難易度・AIツールフィルター
│   ├── PromptCard.tsx       # 一覧カード
│   ├── ReviewCard.tsx       # 成功事例カード（役に立ったボタン付き）
│   ├── ForkCard.tsx         # 派生プロンプトカード
│   └── ReviewModal.tsx      # レビュー投稿モーダル
├── data/
│   └── prompts.json         # ダミーデータ（15件 + レビュー・Fork付き）
├── types/
│   └── index.ts             # 型定義（Prompt / Review / Fork）
└── ...設定ファイル
```

## 主要機能

- **一覧ページ**：カテゴリ・難易度・AIツール・キーワードで絞り込み。成功事例数順/使用数順/新着順でソート
- **詳細ページ**：プロンプト本文コピー、変数説明、使い方ヒント、成功事例レビュー、派生プロンプト
- **レビュー投稿**：職種・業種・AIツール・成果・カスタマイズ・評価を投稿（ローカルstateに反映）
- **役に立ったボタン**：各レビューにワンクリック評価

## カラーパレット

| 用途 | 色 |
|------|-----|
| メイン（信頼・実績） | Amber (#d97706) |
| 背景・テキスト | Slate Navy (#1e293b) |
| 派生プロンプト | Teal (#0d9488) |
| 背景 | Warm off-white (#f8f7f4) |

一般的なAIツールの紫グラデーションを避け、ビジネスユーザーが信頼を感じるAmber+Slateパレットを採用。
